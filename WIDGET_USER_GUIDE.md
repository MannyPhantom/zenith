## Common Questions

### Q: Will my layout save if I close the browser?
**A:** Yes! Your layout is saved in browser storage and persists across sessions.

### Q: Can I have different layouts on different computers?
**A:** Yes, layouts are stored per browser, so each computer/browser can have its own layout.

### Q: What if I mess up my layout?
**A:** Just click "Reset to Default Layout" in Widget Settings to start over.

### Q: Can I add new widgets?
**A:** New widgets added by the development team will automatically appear in your Widget Settings.

### Q: Does it work on mobile?
**A:** Yes! The drag-and-drop feature works on touch devices too. Tap and hold to drag.

### Q: Can I share my layout with others?
**A:** Currently layouts are stored locally. Future versions may support sharing.

## Keyboard Shortcuts

- **Tab** - Navigate between widgets
- **Enter** - Activate drag (when focused on grip handle)
- **Arrow Keys** - Move widget (when dragging)
- **Escape** - Cancel drag operation

## Troubleshooting

### Drag not working?
- Make sure you're in Edit Mode (click "Customize Layout")
- Click the grip icon (ğŸ¯), not the widget itself
- Try refreshing the page

### Layout not saving?
- Check that your browser allows localStorage
- Disable private/incognito mode
- Try a different browser

### Widgets disappeared?
- Open Widget Settings and check if they're toggled off
- Click "Reset to Default Layout" to restore all widgets
